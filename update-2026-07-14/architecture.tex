    \footnotesize
    \textbf{Encode once, then decide.}
    \begin{itemize}\setlength\itemsep{3pt}
      \item \textbf{Cross first:} each queued job attends the \emph{running}
            jobs (soon-to-free nodes) $+$ an \emph{always-valid ctx token} --
            release-aware, and defined even with \emph{no running jobs}.
      \item \textbf{Self next ($N\times$):} the queue self-attends, scoring
            each job \emph{relative to the rest}.
      \item \textbf{Two action heads:} a \textbf{pointer} MLP (shared per job)
            gives one logit per job; a \textbf{no-op} MLP pools the jobs into
            action-0. Joined in one softmax over $V_q{+}1$.
      \item \textbf{Masking:} padded / infeasible jobs $\to -\infty$ pre-softmax,
            so only feasible jobs \& the no-op compete.
      \item \textbf{Critic:} a \textbf{value} MLP scores $\hat V(s)$ for PPO.
    \end{itemize}
